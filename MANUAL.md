# Manual

### Keys and Mousebuttons
**WASD** - Movement
**Leftclick** - Interact
**C** - Crafting
**I** - Inventory
**P** - Character Customisation
**/** - Save the game

### Other
If you wish to start afresh after saving the game, simply quit the game, delete "save.state" and relaunch.

### Debug
**J** - Toggle Trade
